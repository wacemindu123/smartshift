## Demo Flow (15 minutes)

### 1. Manager Dashboard (2 min)
**Show:** `/dashboard`
- [ ] Point out weekly calendar with color-coded coverage
- [ ] Show today's attendance (who's working now)
- [ ] Highlight notification count

**Say:** *"This is your command center. See the whole week at a glance - green means you're staffed well, yellow means you need more people."*

---

### 2. Weekly Schedule (3 min)
**Show:** `/schedule`
- [ ] Show 74 shifts across 2 weeks
- [ ] Point out morning/evening shift patterns
- [ ] Show how to create a new shift
- [ ] Demonstrate editing a shift

**Say:** *"You've got 74 shifts scheduled. Morning crew 7-3, evening 3-11. Click any shift to edit it. Drag to move it around."*

---

### 3. Time-Off Requests (2 min)
**Show:** `/time-off`
- [ ] Show Sarah's pending vacation request
- [ ] Show Mike's approved request
- [ ] Demonstrate approving Sarah's request
- [ ] Show denial option with reason

**Say:** *"Sarah wants 3 days off next month. One click to approve or deny. No more text messages back and forth."*

---

### 4. Shift Swaps (3 min)
**Show:** `/shift-swaps`
- [ ] Show pending swap request
- [ ] Explain 3-step process (request â†’ claim â†’ approve)
- [ ] Demonstrate approving a swap
- [ ] Show how it updates the schedule

**Say:** *"When someone can't make their shift, they request a swap. Another employee claims it. You approve it. You're always in control."*

---

### 5. Team Management (2 min)
**Show:** `/team`
- [ ] Show 8 demo employees
- [ ] Point out assigned roles
- [ ] Show contact info (phone for SMS)

**Say:** *"Here's your whole team. Everyone has a role - barista, register, prep, etc. When you publish schedules, they get notified."*

---

### 6. Capacity View (2 min)
**Show:** `/capacity`
- [ ] Show staffing levels by day
- [ ] Point out Front/Back/Expediter breakdown
- [ ] Highlight any understaffed days

**Say:** *"This shows if you're over or understaffed. You set the max - default is 7. It warns you if you're scheduling too many people."*

---

### 7. Employee View (1 min - Optional)
**Show:** `/my-shifts` (as employee)
- [ ] Show assigned shifts
- [ ] Demonstrate clock in/out
- [ ] Show "Request Swap" button

**Say:** *"Your employees see their shifts, clock in from their phone, and can request swaps. Everything's in one place."*
